## Клавиши:  
| Клавиша | Действие |
|---------|----------|
| Пробел | Пауза |
| R | Пересоздать мир |
| C | Смерть от старости вкл/выкл |
| X | Отрисовка вкл/выкл |
| S | Сохранить |
| L | Загрузить |
| + | Ускорить |
| - | Замедлить |
| E | Режим отображения |
| ЛКМ | Копировать бота |
| ПКМ | Вставить бота |
