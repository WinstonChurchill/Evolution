# Evolution

Тут будет описание проекта

## Использование
### Управление и клавиши:  

| Клавиша | Действие |
|---------|----------|
| `Пробел` | Пауза |
| `R` | Пересоздать мир |
| `C` | Смерть от старости вкл/выкл |
| `X` | Отрисовка вкл/выкл |
| `S` | Сохранить |
| `L` | Загрузить |
| `+` | Ускорить |
| `-` | Замедлить |
| `E` | Режим отображения |
| `ЛКМ` | Копировать бота |
| `ПКМ` | Вставить бота |

## Разработка и развертывание

### Требования:
* **Интерпретатор**: [Python 3.12.10](https://www.python.org/downloads/release/python-31210/)
* **Зависимости**: <!-- Актуальные версии библиотек зафиксированы в pyproject.toml -->
  * `numpy 2.4.1`
  * `pygame 2.5.2`

### Установка зависимостей:
**Windows (PowerShell):**
```powershell
.\installation\install.ps1
```

**macOS / Linux (Bash):**
```bash
./installation/install.sh && source .venv/bin/activate
```

### Компиляция проекта (сборка в .exe/.app): 
```
pyinstaller --onefile --distpath . --icon=gfx/1.ico --name Evolution main.py
```
